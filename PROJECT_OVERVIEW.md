# ICON - In Case Of Need Information Manager

## About This Project

ICON is an **In Case Of Need** application designed for families and close friends to securely store and retrieve critical information during emergencies.

### Categories Supported

- **LEGAL**: Lawyer contacts, Will, Power of Attorney
- **HEALTH**: Doctor details, Carer information  
- **FINANCE**: Bank accounts, Pensions, Loans, ISAs, Mortgages
- **SERVICES**: Utilities, Energy, Telecoms
- **INSURANCE/ASSURANCE**: Car, House, Personal, Warranties, Guarantees, Life insurance, Policy references
- **MEMBERSHIPS**: Subscriptions, Memberships, References

### Key Features

- **Web-based authentication** for secure access
- **Upload documents** (scans, PDFs) alongside text entries
- **Owner controls** - only you can edit/delete
- **Family sharing** - limited view access
- **Local storage** - runs on your machine with SQLite
- **Responsive design** - works on desktop and tablets
